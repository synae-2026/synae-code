# SynAE Code

This repository contains the code for the paper "SynAE: A Framework for Measuring the Quality of Synthetic Data for Agent Evaluations". 

## Contents

1. SynDataGeneration: Code to create synthetic benchmarks via interpretable generation algorithms as described in the paper. 
2. Benchmarks: Original benchmark code with helper scripts to generate synthetic tool calls/outputs, and run agents on both original and synthetic benchmarks. 
3. SynAE_Evaluation: Code that evaluates the quality of synthetic benchmarks using the SynAE Framework. 

## Requirements

Python 3.11

Each folder has a README that describes how to setup the code and run it.
